ポートフォリオサイト

  • 制作時間

    96時間

  • 使用ツール

    illustrator　html　css　javascript

詳細

就職活動用のポートフォリオサイトです。 ファーストビューにはカフェ巡りが好きなことを知ってもらいたかった為、過去に撮った写真を貼り付けました。 自己紹介欄のところは友達や親に私のイメージカラーを聞いた際、黄色と答えてもらう事が多かったので黄色にしました。 それに合わせて、他の部分の色は黄色に合う色を選び茶色や黄色みの白を配色しました。 サイトを作る際、工夫したところが二つあります。 一つ目はナビの部分です。スライドした時にナビが動くようになっており、見たい情報のところにすぐに飛べるようにしました。 二つ目は作品の部分です。カテゴリーごとに作品を分け、見た時にすぐにわかるようにしました。また、リンクのついている作品は文字にカーソルを 合わせた際、色が変わるようにしています。
